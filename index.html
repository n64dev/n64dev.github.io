Useful N64 Hacking Links

Emulators

  • CEN64 — a cycle-accurate N64 emulator. development blog
  • MAME — accurate but slow
  • Project64 — fast but sacrifices accuracy for performance
  • Mupen64Plus — a cross-platform plugin-based N64 emulator
  • 64DD emulation news

Software

  • n64chain — Tyler Stachecki's N64 toolchain
  • libdragon — DragonMinded's N64 library
  • Open Source Nintendo 64 — massive git repo containing neon64, gsuploader, libn64, and many docs
  • n64crc — C program to fix cart header checksums
  • leotools — extract files from 64DD disk images
  • agbd Game Boy dumper — dump Game Boy ROMs and saves using transfer pak

Reference

  • N64 Bare Metal Mips Assembly Programming — krom's excellent tests and sample programs
  • vru.txt — DragonMinded's description of the VRU
  • Registers — N64 CPU register reference
  • krom's patent list — big list of N64-related patents, also mirrored locally here
  • #n64dev — N64 development IRC channel on EFnet
  • Vr4300, Vr4305, Vr4310 User Manual
  • N64 Games And Hardware You Didn't Know About — a ton of pictures and some info

Presentations

  • Reversing the N64 CIC — REcon 2015 by Mike Ryan, marshallh, and John McMaster
    • UltraCIC — open source N64 CIC clone
    • sm5emu — SM5 CPU emulator
